# Dice Images Directory

Place your high-quality dice images here:

- `dice-1.png`
- `dice-2.png`
- `dice-3.png`
- `dice-4.png`
- `dice-5.png`
- `dice-6.png`

## Image Requirements

- Format: PNG (with transparency recommended)
- Size: Square, ideally 400x400px or larger
- Style: Match the glossy white dice with black dots style from your reference image

## Using Your Reference Image

If you have the dice image from ChatGPT, you can:
1. Crop each die individually
2. Save as `dice-1.png` through `dice-6.png`
3. Place them in this directory

The app will automatically load and display them!
